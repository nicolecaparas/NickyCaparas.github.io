2.) Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'. (But not those ones):

Loops allow us to repeat code if it satisfies a set of condition/s. There are different types of loops: the for loop, while loop, for..of loop, and the for..in loop. If your condition returns true, your code block wll run, and then the condition will be checked again and if it still returns true, the code block will run again. It repeats until the condition returns false. It is like wearing your FitBit or Apple Watch: If you start targeting or reaching your exercise goals it will continue to count the steps/movements on your watch. The only time it will stop is once you remove your watch or not even do any movements for that day.

3.)Describe what the DOM is and an example of how you might interact with it:

DOM specifies how browsers should create a model of an HTML page and how JavaScript can access and update the contents of a web page while it is in the browser window. The DOM has separate set of rules which are implemented by all major browser makes and covers two primary areas: (a) Making a model of the HTML Page; (b) Accessing and changing the HTML page.

As you work with the DOM tree you need to:

A. Locate the node that represents the element you want to work with:

  • Selecting an individual element node: getElementById() / querySelector();
  • Selecting multiple elements (nodelists): getElementByClassName() / getElementByTagName()/ querySelectorAll();
  • Traversing between element nodes: parentNode / firstChild or lastChild/ previousSibling or nextSibling;

B.Use its text content, child elements and attributes.

  • Access/ update text nodes: nodeValue
  • Work with HTML Content: innerHTML / textContent / CreateElement()or CreateTextNode()or appendChild() /removeChild()
  • Access or update attribute values: hasAttribute() / getAttribute()/ setAttribute() / removeAttribute()

4.)Explain the difference between accessing data from arrays and objects:

Array
Array is a special data type in JavaScript. It doesn't just store one value -- it stores a list of values. It is an ordered collection of values similar to the songs in a playlist: When you create a playlist of songs you have different methods you can do like: delete, add, replay, play from start to end songs and many more. Just like an array there are different methods as well to access your data:

  • push : add an array to end
  • pop : remove an array from the end
  • shift: remove an array from the start
  • unshift: add array to the start
  • concat: merge array
  • includes: look for a value
  • indexOf: searches for a value inside the array and returns its index.
    Returns -1 if the value does not exist.
  • join: creates a string from a array
  • reverse: reverses an array
  • slice: copies a portion on an array
  • splice: removes/ replaces elements
  • sort: sorts an array

Objects
Objects are also a special data type in JavaScript. Just like arrays, objects are going to help store multiple pieces of data together in some sort of structure. But unlike arrays where the whole point is to order our data, this is not the case with objects. Instead, our data is stored using what we call key-value pairs or properties. So imagine if you have a bag of gems with different property types (like shape, size, color, type, etc.): These pieces of gems have no order but they are in one bag that are appealing and amazing to look at and its easy to take remove or add from the bag when you know the type you're looking for. This is how similar objects are when you store and access data. These data need access key to get these bits of information with correct label on it.

5.)Explain what functions are and why they are useful:

Functions allow us to write reusable, modular code. We define a "chunk" of code that we can then execute at a later point.We use them all the time and its 2-step process which is to "define it" and then "run with it". A good analogy would be a washer-dryer combo appliance. You put in clothes inside of it, and apart from washing the clothes, it also dries them after it finishes washing them, saving you time.
